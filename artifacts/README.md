# Artifacts Layout (Task-Oriented)

- `classification/learning/`: learning-based classification outputs
- `classification/traditional/`: traditional classification outputs
- `detection/gen1/`: GEN1 detection outputs
- `optical_flow/mvsec/`: MVSEC optical-flow outputs
- `cross_task_reports/latest/`: latest integrated comparison reports
- `archive/`: historical snapshots and old report generations

- `artifacts/learning_classification`
- `artifacts/traditional_classification`
